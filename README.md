## 概要 h2
東京の物件情報が入ってこない地方出身者のための不動産サイト．

## コンセプト h2
simple　×　easy　×　Region

## バージョン h2
Ruby 2.4.0 Rails 5.1.1

## 機能一覧 h2
- ログイン機能
-	ユーザー登録機能
  - 名前，メールアドレス，パスワード
-	物件一覧表示機能
-	物件登録機能
  -	物件情報，写真
-	物件編集機能
  -	登録者のみ編集可能
-	物件削除機能
  -	登録者のみ削除可能
-	検索機能
-	お問い合わせ機能
  -	利用者から不動産会社
-	相場比較機能
